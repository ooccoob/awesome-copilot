# PCF 社区资源

Power Apps 组件框架拥有一个充满活力的社区，用于创建和共享资源、工具和知识。本指南提供了关键社区资源的链接。

## PCF画廊

**[PCF 画廊](https://pcf.gallery)**

PCF Gallery 是发现、共享和学习 PCF 组件的中心枢纽。

**您会发现什么：**
- 社区创建的组件
- 组件演示和屏幕截图
- 源代码链接
- 安装说明
- 组件评级和评论

**使用方法：**
- 按类别浏览组件
- 搜索特定功能
- 下载适合您项目的组件

- 从现实世界的组件实现中学习

## 社区最佳实践

1. **分享您的工作**：向社区贡献组件和知识
2. **提供反馈**：报告问题并提出改进建议
3. **文档齐全**：在您的组件中包含清晰的文档
4. **彻底测试**：在共享之前确保组件跨平台工作
5. **遵循标准**：使用既定的模式和命名约定
6. **乐于助人**：回答问题并帮助其他开发人员

## 为 PCF 画廊做出贡献

要将您的组件添加到 PCF Gallery：

1. 创建一个文档齐全的组件
2. 跨目标平台测试
3. 准备截图和演示
4. 提交至 pcf.gallery
5. 包含源代码链接（推荐 GitHub）
6. 提供清晰的安装说明
