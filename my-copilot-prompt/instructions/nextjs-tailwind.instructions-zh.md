# Next.js + Tailwind 开发说明

使用 Tailwind CSS 样式和 TypeScript 的高质量 Next.js 应用程序的说明。

## 项目背景

- 最新的 Next.js（应用程序路由器）
- 用于类型安全的 TypeScript
- Tailwind CSS 样式

## 开发标准

### 建筑
- 具有服务器和客户端组件的 App Router
- 按功能/域对路由进行分组
- 实施适当的错误边界
- 默认使用 React 服务器组件
- 尽可能利用静态优化

### 打字稿
- 启用严格模式
- 清晰的类型定义
- 使用类型保护进行正确的错误处理
- Zod 用于运行时类型验证

### 造型
- Tailwind CSS 具有一致的调色板
- 响应式设计模式
- 深色模式支持
- 遵循容器查询最佳实践
- 维护语义 HTML 结构

### 状态管理
- 用于服务器状态的 React 服务器组件
- 客户端状态的 React hooks
- 正确的加载和错误状态
- 适当的乐观更新

### 数据获取
- 用于直接数据库查询的服务器组件
- React Suspense 加载状态
- 正确的错误处理和重试逻辑
- 缓存失效策略

### 安全性
- 输入验证和清理
- 正确的身份验证检查
- 跨站请求伪造保护
- 速率限制实施
- 安全 API 路由处理

### 性能
- 使用 next/image 进行图像优化
- 使用 next/font 进行字体优化
- 路由预取
- 正确的代码分割
- 捆绑包大小优化

## 实施流程
1. 规划组件层次结构
2. 定义类型和接口
3. 实现服务器端逻辑
4. 构建客户端组件
5. 添加适当的错误处理
6. 实施响应式样式
7. 添加加载状态
8. 编写测试
