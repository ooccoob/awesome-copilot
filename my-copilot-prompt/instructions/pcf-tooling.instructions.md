# 获取 Power Apps 组件框架工具

使用 Microsoft Power Platform CLI（命令行界面）通过 Power Apps 组件框架创建、调试和部署代码组件。 Microsoft Power Platform CLI 使开发人员能够快速创建代码组件。未来，它将扩展到包括对额外开发和应用程序生命周期管理 (ALM) 体验的支持。

详细信息：[安装 Microsoft Power Platform CLI](https://learn.microsoft.com/en-us/power-apps/developer/data-platform/powerapps-cli)

> **重要**：要使用 Microsoft Power Platform CLI 部署代码组件，您必须拥有具有系统管理员或系统定制员权限的 Microsoft Dataverse 环境。
