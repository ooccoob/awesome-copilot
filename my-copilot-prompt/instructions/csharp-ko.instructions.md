# C# 编程语言

## 명명 규칙（命名约定）

일관된 명명 규칙은 코드 і독성의 핵심입니다。 Microsoft 已为您提供了解决方案。

| 요소 | 명명 규칙 | 예시 |
|------|-----------|------|
| 인터페기스 | 접두사 'I' + PascalCase | __代码0__，__代码1__ |
| 공개(公共) 멤버 | 파스칼 케 Beeni스 (PascalCase) | __代码0__，__代码1__ |
| 매개변수, 지역 변수 | 카멜 케֊（驼峰命名法）| __代码0__，__代码1__ |
| 비공개/내부 필드 | 밑줄(_) + 카멜 케기스 | __代码0__ |
| 상수 (常量) | 파스칼 케 Beeni스 (PascalCase) | __代码0__ |
| 제네릭 형식 매개변수 | 접두사 'T' + 설명적인 름 | __代码0__、__代码1__、__代码2__ |
| 비동기 메서드 | '异步' 접미사 | __代码0__，__代码1__ |
