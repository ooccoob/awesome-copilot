# 7400 系列逻辑 IC 规格

## 1. 概述

**7400 系列**是一个庞大的**数字逻辑集成电路**系列，最初以**TTL（晶体管-晶体管逻辑）**实现，后来扩展到包括**CMOS 兼容变体**。这些器件提供基本构建模块，例如门、触发器、计数器、多路复用器、解码器和总线收发器。

7400 系列 IC 广泛应用于**逆向计算**、**6502/Z80 系统**、**胶合逻辑**和教育设计。

---

## 2. 逻辑族

|家庭|技术 |笔记|
| ------- | ----------------------- | ----------------------- |
| 74xx | TTL |原装双极TTL |
| 74LSxx |低功耗肖特基 TTL |速度更快、功耗更低 |
| 74HCTxx | CMOS（TTL电平输入）|混合系统的理想选择 |
| 74HCxx | CMOS |宽电压范围 |
| 74ACTxx |先进的CMOS TTL电平|非常快|

---

## 9. 接口说明

* TTL 输出可靠地驱动 TTL 输入
* CMOS 输入不得浮动
* 将 CMOS 与 TTL 信号连接时使用 **74HCT**
* 每个 IC 需要去耦电容器 (0.1 µF)
