# AS6C62256 32K x 8 低功耗 CMOS SRAM 规格

## 1. 概述

**AS6C62256** 是一款 **256 Kbit (32K x 8)** 低功耗 CMOS 静态随机存取存储器 (SRAM)，由 Alliance Memory 制造（由其他公司二次采购）。它与常见的 28 引脚 SRAM 引脚排列完全兼容，广泛应用于 **6502**、**Z80** 和其他 8 位微处理器系统的读/写存储器。

该器件提供快速的访问时间、简单的控制逻辑和极低的待机功耗。

## 3. 引脚配置（逻辑）

### 3.1 地址引脚（A0-A14）

* 选择 32,768 个内存位置之一
* 地址在读或写周期内必须稳定

### 3.2 数据引脚（I/O0-I/O7）

* 双向三态数据总线
* 未启用时为高阻抗

### 3.3 控制引脚

|针 |描述 |
| --- | -------------------------- |
| CE# |芯片使能（低电平有效）|
| OE# |输出使能（低电平有效）|
|我们# |写使能（低电平有效）|
| VCC | +5V电源|
|接地 |地面|

---

## 4. 内存组织

* 从 `$0000` 到 `$7FFF` 的线性地址空间
* 每个地址对应一个字节（8位）

```text
Address range: 0000h - 7FFFh
Data width:    8 bits
```

## 6. 写操作

|行政长官？    |低|

## 8. 电源模式

### 8.1 主动模式

* 行政长官？ = 低
* 正常读/写操作

### 8.2 待机模式

* 行政长官？ = 高
* 保留数据
* 功耗极低

---

## 9. 复位和上电行为

* 无需复位引脚
* 上电时数据未定义
* 仅当 VCC 存在时才保留存储器内容
