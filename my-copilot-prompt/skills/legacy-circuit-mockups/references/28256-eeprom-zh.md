# AT28C256 256K (32K x 8) 并行 EEPROM 规格

## 1. 概述

**AT28C256** 是 Atmel（现为 Microchip）制造的非易失性、电可擦除可编程只读存储器 (EEPROM)。它提供 **256 Kbits** 的存储，组织为 **32,768 x 8 位**，通常用于 8 位微处理器系统，例如基于 **6502**、**Z80** 和类似 CPU 的系统。

该器件支持字节级写入操作、快速读取访问和软件控制的数据保护。

---

|特色|描述 |

## 3. 引脚配置（逻辑）

### 3.1 地址引脚（A0-A14）

* 选择 32,768 个内存位置之一

### 3.2 数据引脚（I/O0-I/O7）

* 双向三态数据总线
* 读周期期间输出有效

### 3.3 控制引脚

|针 |描述 |
| --- | -------------------------- |
|欧盟CE |芯片使能（低电平有效）|
|原厂设备 |输出使能（低电平有效）|
|我们|写使能（低电平有效）|
| VCC | +5V电源|
|接地 |地面|

---

## 4. 内存组织

* 从 `$0000` 到 `$7FFF` 的线性地址空间
* 每个地址对应一个8位字节

```text
Address Range: 0000h - 7FFFh
Data Width:    8 bits
```

---

## 5. 读操作

### 5.1 读周期条件

|信号|状态|
| ------ | ----- |
|欧盟CE |低|
|原厂设备 |低|
|我们|高|

* 访问时间后数据出现在 I/O 引脚上
* CE 和 OE 置位时输出保持有效
* 当 CE 或 OE 为高电平时，输出为高阻抗

---

## 6. 写操作

### 6.1 字节写周期

|信号|状态|
| ------ | --------- |
|欧盟CE |低|
|原厂设备 |高|
|我们|低脉冲|

* 地址和数据在 WE 低脉冲期间必须稳定
* 内部写入周期时间 ≈ 10 ms（最大值）
* 设备自动处理写入前擦除

---

## 7. 软件数据保护（SDP）

AT28C256 包括可选的**软件数据保护**，以防止意外写入。

### 7.1 SDP 使能序列

```text
Write $AA to address $5555
Write $55 to address $2AAA
Write $A0 to address $5555
```

### 7.2 SDP 禁用序列

```text
Write $AA to address $5555
Write $55 to address $2AAA
Write $80 to address $5555
Write $AA to address $5555
Write $55 to address $2AAA
Write $20 to address $5555
```

---

## 8. 写入周期时序注释

* 写入是内部定时的；无需外部轮询
* 在写周期期间，读取返回未定义的数据
* 设备在繁忙时忽略额外的写入尝试

---

## 9. 数据轮询（可选）

* I/O7 可以在写入期间被监视
* 当I/O7与写入数据匹配时，写入完成

---

## 10. 复位和电源行为

* 没有明确的复位引脚
* 加电和断电期间禁止写入
* 在 CE 和 OE 置位之前，输出默认为高阻抗
