# 常见面包板电子元件

**基于面包板的电子产品**的实用参考，涵盖常见组件、公式、表格、引脚排列和学习参考。适合初学者到中级硬件构建者。

---

## 1. 电阻器

### 描述

电阻器限制电流、分压并设置偏置点。

### 常见类型

|类型 |笔记|典型用途|
| ------------- | ----------- | ------------------ |
|碳膜|便宜，±5% |通用|
|金属膜|稳定，±1% |精密电路|
|线绕|高功率|功耗|
|电位器|可调|音量、调音|

### 标准值（E12系列）

__代码0__

### 颜色代码

|乐队 |意义|
| ---- | ------------ |
| 1 |第一个数字 |
| 2 |第二个数字 |
| 3 |乘数|
| 4 |公差|

### 额定功率

|评级 |典型尺寸|
| ------ | ------------------- |
| 1/8 瓦 |小|
| 1/4 瓦 |面包板标准|
| 1/2 瓦 |大|

### 公式

* **欧姆定律：** `V = I x R`
* **电源：** `P = V × I = I² × R = V² / R`

### 参考

* [https://en.wikipedia.org/wiki/Resistor](https://en.wikipedia.org/wiki/Resistor)

## 2. 电容器

### 描述

电容器存储电能并阻止直流电，同时传递交流电。

### 常见类型

|类型 |偏光|典型用途|
| ------------ | --------- | ------------------ |
|陶瓷|没有 |解耦、旁路|
|电解 |是的 |批量过滤 |
|钽|是的 |紧凑型过滤 |
|电影|没有 |信号耦合|

### 共同价值观

|范围 |示例|
| ----- | ---------------- |
| pF | 10pF、100pF |
| nF| 100nF（0.1μF）|
| µF | 1μF、10μF、100μF |

### 电压额定值规则

> 电容器额定电压 ≥ **2x 电路电压**

### 公式

* **容抗：** `Xc = 1 / (2πfC)`
* **储存能量：** `E = ½ C V²`

### 参考

* [https://en.wikipedia.org/wiki/Capacitor](https://en.wikipedia.org/wiki/Capacitor)

## 3. 电感器

### 描述

电感器在磁场中存储能量并抵抗电流的变化。

### 常见类型

|类型 |使用|
| ------------ | ----------------- |
|空心 |射频电路|
|铁氧体磁芯 |电源滤波|
|窒息|噪音抑制|

### 公式

* **感抗：** `XL = 2πfL`

### 参考

* [https://en.wikipedia.org/wiki/Inductor](https://en.wikipedia.org/wiki/Inductor)

## 4. 二极管

### 描述

二极管仅允许电流沿一个方向流动。

### 常见类型

|类型 |典型部分|使用|
| --------- | ------------ | --------------------- |
|整流器| 1N4007 |电源|
|信号| 1N4148 |逻辑，快速切换|
|齐纳| 1N4733A |电压调节|
|肖特基| 1N5819 |低落|

### 关键参数

|参数|典型|
| --------------- | -------------------------- |
|正向电压| 0.7V（硅）、0.3V（肖特基）|
|反向电压|特定于设备 |

## 7. 集成电路（DIP）

### 通用逻辑族

## 9. 面包板

### 描述

具有内部总线连接的无焊原型板。

### 内部接线

* 排数：水平连接（5孔）
* 导轨：垂直连接（电源）

### 局限性

* 不适合高频或大电流电路
