# 基础电子元件

本文档涵盖四种基本电子元件：电阻器、电容器、按钮和开关，包括常用值、公式和类型。

---

## 1. 电阻器（R）

电阻器限制电流并以热量的形式耗散能量。

### 常用公式

* **欧姆定律：** $V = I \times R$（电压=电流$\times$电阻）
* **串联电阻：** $R_{总计} = R_1 + R_2 + ... + R_n$
* **并联电阻：** $\frac{1}{R_{total}} = \frac{1}{R_1} + \frac{1}{R_2} + ... + \frac{1}{R_n}$

### 通用值（E12/E24 系列 - 5% 容差）

值乘以 10 的幂（例如，10$\Omega$、100$\Omega$、1k$\Omega$、10k$\Omega$、100k$\Omega$、1M$\Omega$）：
**1.0, 1.1, 1.2, 1.3, 1.5, 1.6, 1.8, 2.0, 2.2, 2.4, 2.7, 3.0, 3.3, 3.6, 3.9, 4.3, 4.7, 5.1, 5.6, 6.2, 6.8, 7.5, 8.2, 9.1**

---

## 2. 电容器（C）

电容器将电荷存储在电场中。

### 常用公式

* **电容：** $C = \frac{Q}{V}$（电荷/电压）
* **当前：** $i = C \frac{dV}{dt}$
* **并联电容：** $C_{总计} = C_1 + C_2 + ... + C_n$
* **串联电容：** $\frac{1}{C_{total}} = \frac{1}{C_1} + \frac{1}{C_2} + ... + \frac{1}{C_n}$
* **储存能量：** $W = \frac{1}{2} C V^2$

### 共同价值观

* **陶瓷（皮法/纳法）：** 10pF、22pF、100pF、1nF、10nF、100nF（通常标记为“104”）
* **电解（微法）：** 1μF、10μF、22μF、47μF、100μF、220μF、470μF、1000μF

---

## 3. 按钮（按钮）

一种瞬时开关，仅在按下时才完成电路。

### 常见类型

* **常开 (NO)：** 电路打开（关闭）直至按下。
* **常闭 (NC)：** 电路闭合（打开），直到按下为止。
* **轻触开关：** 用于 PCB 安装的小型标准按钮。

---

## 4. 开关（SW）

一种机电装置，可断开或连接电路，并保持原位直至翻转。

### 常见类型

* **SPST：** 单刀单掷（开/关）
* **SPDT：** 单刀双掷（在两个路径之间切换）
* **DPST/DPDT：** 双极变型（同时控制两个独立电路）
* **DIP 开关：** 用于电路板的小型开关阵列。
