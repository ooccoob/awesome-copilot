# 目视检查清单

本文档是网页设计目视检查期间要验证的项目的综合清单。

---

## 1. 布局验证

### 结构完整性

- [ ] 标题正确固定/位于屏幕顶部
- [ ] 页脚位于屏幕底部或内容末尾
- [ ] 主要内容区域居中对齐，宽度适当
- [ ] 侧边栏（如果有）位置正确
- [ ] 导航显示在预期位置

### 溢出

- [ ] 水平滚动条不会无意显示
- [ ] 内容不会从父元素溢出
- [ ] 图像适合父容器
- [ ] 表格不超过容器宽度
- [ ] 代码块适当换行或滚动

### 对准

- [ ] 网格项均匀分布
- [ ] Flex 项目对齐正确
- [ ] 文本对齐（左/中/右）一致
- [ ] 图标和文本垂直对齐
- [ ] 表单标签和输入字段位置正确

---

## 2. 版式验证

### 可读性

- [ ] 正文字体大小足够（建议最小 16px）
- [ ] 线高合适（推荐1.5-1.8）
- [ ] 每行字符数为宜（建议40-80个字符）
- [ ] 段落之间的间距足够
- [ ] 标题大小层次结构清晰

### 文本处理

- [ ] 长单词适当换行
- [ ] URL 和代码处理正确
- [ ] 不发生文本剪切
- [ ] 省略号 (...) 显示正确
- [ ] 特定于语言的换行规则正常工作

### 字体

- [ ] 网络字体正确加载
- [ ] 后备字体是合适的
- [ ] 字体粗细符合预期
- [ ] 特殊字符和表情符号正确显示

---

## 3. 颜色和对比度验证

### 辅助功能（WCAG 标准）

- [ ] 正文：对比度 4.5:1 或更高 (AA)
- [ ] 大文本（18px+ 粗体或 24px+）：3:1 或更高
- [ ] 交互元素边框：3:1 或更高
- [ ] 焦点指示：与背景有足够的对比度

### 颜色一致性

- [ ] 品牌颜色统一
- [ ] 链接颜色一致
- [ ] 错误状态红色统一
- [ ]成功状态绿色统一
- [ ] 悬停/活动状态颜色合适

### 色觉多样性

- [ ] 通过形状和文字传达信息，而不仅仅是颜色
- [ ] 图表和图表考虑了色觉多样性
- [ ] 错误消息不仅仅依赖于颜色

---

## 4. 响应式验证

### 移动设备（~640 像素）

- [ ] 内容适合屏幕宽度
- [ ] 触摸目标为 44x44px 或更大
- [ ] 文字大小可读
- [ ] 不发生水平滚动
- [ ] 导航适合移动设备（汉堡菜单等）
- [ ] 表单输入易于使用

### 平板电脑 (641px~1024px)

- [ ] 布局针对平板电脑进行了优化
- [ ] 两列布局正确显示
- [ ] 图像尺寸合适
- [ ] 侧边栏显示/隐藏是否合适

### 桌面（1025px~）

- [ ] 最大宽度已设置并且在超大屏幕上不会中断
- [ ] 间距足够
- [ ] 多列布局功能正常
- [ ] 悬停状态已实现

### 断点转换

- [ ] 屏幕尺寸变化时布局平滑过渡
- [ ] 布局在中间尺寸下不会中断
- [ ] 内容不会消失或重复

---

## 5. 交互元素验证

### 按钮

- [ ] 默认状态清除
- [ ] 存在悬停状态（桌面）
- [ ] 对焦状态视觉清晰
- [ ] 存在活动（按下）状态
- [ ] 禁用状态可区分
- [ ] 加载状态（如果适用）

### 友情链接

- [ ] 链接清晰可辨
- [ ] 访问过的链接是可区分的（如果需要）
- [ ] 存在悬停状态
- [ ] 对焦状态清晰

### 表单元素

- [ ] 输入字段边界清晰
- [ ] 占位文字对比度合适
- [ ] 焦点视觉反馈
- [ ] 错误状态显示
- [ ] 必填字段指示
- [ ] 下拉菜单功能正常

---

## 6. 图像和媒体验证

### 图片

- [ ] 图像以适当的尺寸显示
- [ ] 保持纵横比
- [ ] 高分辨率显示支持 (@2x)
- [ ] 图片加载失败时显示
- [ ] 延迟加载行为有效

### 视频和嵌入

- [ ] 视频适合容器
- [ ] 保持纵横比
- [ ] 嵌入内容是响应式的
- [ ] iframe 不会溢出

---

## 7. 可访问性验证

### 键盘导航

- [ ] 所有交互元素均可通过 Tab 键访问
- [ ] 焦点顺序合乎逻辑
- [ ] 焦点陷阱是否合适（情态动词等）
- [ ] 存在跳转到内容的链接

### 屏幕阅读器支持

- [ ] 图像有替代文本
- [ ] 表单有标签
- [ ] ARIA 标签已正确设置
- [ ] 标题层次结构正确（h1→h2→h3...）

### 运动

- [ ] 动画不过分
- [ ] 支持偏好减少运动（如果可能）

---

## 8. 与性能相关的视觉问题

### 加载中

- [ ] 字体 FOUT/FOIT 最小
- [ ] 未发生布局偏移 (CLS)
- [ ] 图片加载时不跳转
- [ ] 骨架屏幕合适（如果适用）

### 动画

- [ ] 动画流畅（60fps）
- [ ] 滚动时没有性能问题
- [ ] 过渡自然

---

## 优先级矩阵

|优先|类别 |示例 |
|----------|----------|----------|
| P0（严重）|功能性破坏 |元素完全重叠，内容消失 |
| P1（高）|严重的用户体验问题 |无法阅读的文本、无法操作的按钮 |
| P2（中）|中等问题 |对齐问题、间距不一致 |
| P3（低）|小问题|轻微的定位差异、轻微的颜色变化 |
