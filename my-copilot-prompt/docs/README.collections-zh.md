# 📦 收藏

围绕特定主题、工作流程或用例组织的相关提示、说明和代理的精选集合。
### 如何使用集合

**浏览收藏：**
- ⭐ 精选收藏突出显示并显示在列表顶部
- 探索对相关自定义进行分组的主题集合
- 每个集合都包含特定工作流程的提示、说明和代理
- 集合可以轻松地针对特定场景采用综合工具包

**安装项目：**
- 单击集合中各个项目的安装按钮
- 或者浏览到各个文件以手动复制内容
- 集合可帮助您发现您可能错过的相关定制
