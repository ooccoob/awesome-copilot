## 集合用途

### Java MCP 专家

推荐

此聊天模式为用 Java 构建 MCP 服务器提供专家指导。

此聊天模式非常适合：
- 使用 Java 创建新的 MCP 服务器项目
- 使用 Project Reactor 实现反应式处理程序
- 设置 stdio 或 HTTP 传输
- 调试反应流和错误处理
- 使用官方 SDK 学习 Java MCP 最佳实践
- 与 Spring Boot 应用程序集成

为了获得最佳结果，请考虑：
- 使用指令文件为 Java MCP 开发设置上下文
- 使用提示生成初始项目结构
- 切换到专家聊天模式以获得详细的实施帮助
- 指定是否需要 Maven 还是 Gradle
- 提供有关您需要哪些工具或功能的详细信息
- 提及是否需要 Spring Boot 集成
