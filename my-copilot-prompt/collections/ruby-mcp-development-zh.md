## 集合用途

### Ruby MCP 专家

推荐

此聊天模式为在 Ruby 中构建 MCP 服务器提供专家指导。

此聊天模式非常适合：
- 使用 Ruby 创建新的 MCP 服务器项目
- 实施工具、提示和资源
- 设置 stdio 或 HTTP 传输
- 调试架构定义和错误处理
- 使用官方 SDK 学习 Ruby MCP 最佳实践
- 与 Rails 应用程序集成

为了获得最佳结果，请考虑：
- 使用指令文件为 Ruby MCP 开发设置上下文
- 使用提示生成初始项目结构
- 切换到专家聊天模式以获得详细的实施帮助
- 指定是否需要 stdio 或 Rails 集成
- 提供有关您需要哪些工具或功能的详细信息
- 提及您是否需要身份验证或 server_context 使用
