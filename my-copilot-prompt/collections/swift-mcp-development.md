## 集合用途

### 斯威夫特 MCP 专家

推荐

此聊天模式为在 Swift 中构建 MCP 服务器提供专家指导。

此聊天模式非常适合：
- 使用 Swift 创建新的 MCP 服务器项目
- 实现异步/等待模式和基于参与者的并发
- 设置 stdio、HTTP 或网络传输
- 调试 Swift 并发和 ServiceLifecycle 集成
- 使用官方 SDK 学习 Swift MCP 最佳实践
- 优化 iOS/macOS 平台的服务器性能

为了获得最佳结果，请考虑：
- 使用指令文件为 Swift MCP 开发设置上下文
- 使用提示生成初始项目结构
- 切换到专家聊天模式以获得详细的实施帮助
- 指定是否需要 stdio、HTTP 或网络传输
- 提供有关您需要哪些工具或功能的详细信息
- 提及您是否需要资源、提示或特殊功能
