## 集合用途

### TypeScript MCP 服务器专家

推荐

此聊天模式为在 TypeScript/Node.js 中构建 MCP 服务器提供专家指导。

此聊天模式非常适合：
- 使用 TypeScript 创建新的 MCP 服务器项目
- 通过 zod 验证实施工具、资源和提示
- 设置 HTTP 或 stdio 传输
- 调试模式验证和传输问题
- 学习 TypeScript MCP 最佳实践
- 优化服务器性能和可靠性

为了获得最佳结果，请考虑：
- 使用指令文件为 TypeScript/Node.js 开发设置上下文
- 使用提示生成具有正确配置的初始项目结构
- 切换到专家聊天模式以获得详细的实施帮助
- 指定是否需要 HTTP 或 stdio 传输
- 提供有关您需要哪些工具或功能的详细信息
