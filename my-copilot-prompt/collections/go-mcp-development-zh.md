## 集合用途

### Go MCP 服务器开发专家

推荐

此聊天模式为在 Go 中构建 MCP 服务器提供专家指导。

此聊天模式非常适合：
- 使用 Go 创建新的 MCP 服务器项目
- 使用结构和 JSON 模式标签实现类型安全工具
- 设置 stdio 或 HTTP 传输
- 调试上下文处理和错误模式
- 使用官方 SDK 学习 Go MCP 最佳实践
- 优化服务器性能和并发性

为了获得最佳结果，请考虑：
- 使用指令文件为 Go MCP 开发设置上下文
- 使用提示生成初始项目结构
- 切换到专家聊天模式以获得详细的实施帮助
- 指定是否需要 stdio 还是 HTTP 传输
- 提供有关您需要哪些工具或功能的详细信息
- 提及您是否需要资源、提示或特殊功能
