## 1. 执行摘要
- 总体月度成本、主要驱动因素、潜在节省区间（区间估算，不虚构精确数字）
- TOP N 优化机会（按潜在影响排序）

## 2. 环境概览与分析范围
- 订阅 / 组织层级结构高层视图
- 纳入与排除范围
- 资源数量级分布（计算 / 存储 / 数据库 / 网络 / 安全 / AI 等）

## 3. 成本分布与趋势洞察
- 成本按：服务 / 资源类型 / 资源组 / 环境(Prod/Non-Prod) / 区域 分布
- 趋势模式：线性增长 / 周期性峰值 / 偶发尖刺
- 是否存在季节性或波动特征（若缺数据→假设）

## 4. 异常与异常模式分析
- 突增（服务 / 资源 / 时间窗口）
- 成本基线漂移（>X% 周环或月环）
- 遗留 Orphan 资源（无依赖 / 无流量）
- 未受控实验 / POC 遗留

## 5. 资源级优化发现
为每类资源（VM / AKS / App Service / Function / Batch / Data Factory / Synapse / Cosmos DB / Redis / AI / Monitor 等）列：
```
资源类型 | 发现类型(闲置/低利用/过度/异常) | 典型特征 | 建议措施 | 预期节省(定性) | 备注
```

## 6. 计费与定价优化
- 预留实例 / 节省计划（使用率 / 适配度）评估
- 混合权益 (Azure Hybrid Benefit) 适用性
- 负载是否适合 Spot / 短期突发型
- 是否存在短期弹性需求而被长期计费方案绑定

## 7. 存储与数据层优化
- 存储账户：冷热分层建议
- Blob：访问层不匹配 / 大量未访问对象 / 版本与快照膨胀
- 数据库：性能层级（DTU / vCore） vs 实际负载
- 备份与保留策略成本影响
- 日志保留期限是否过长

## 8. 计算与伸缩优化
- VM/Scale Set：CPU / 内存低利用（<20% 持续）
- AKS：节点池是否过度预配 / 非工作时段缩容缺失
- App Service：AlwaysOn / 计划层级与实例数
- Function：冷启动 vs 预留计划成本平衡
- Autoscale 策略是否与业务节奏匹配

## 9. 网络与流量成本优化
- 出站流量热点区域
- 不必要区域间数据移动
- 是否可使用压缩 / CDN / Cache 降低下游调用成本
- 专线 / VPN / Firewall / WAF 计费评估

## 10. 安全 / 合规引发的成本考量
- Defender / Sentinel / Monitor 采集策略是否精准
- 过度日志采集（Category / 高基数指标）
- 合规要求与冗余保护导致的资源冗余

## 11. 组织与标签治理评估
- 标签覆盖率（环境 / 所有者 / 成本中心）
- 未打标签资源风险
- 是否支持 Showback / Chargeback
- 建议的标签策略增强

## 12. 预留实例 / 节省计划 / 混合权益分析
- 适用范围与利用率评估
- 潜在扩展购买建议（定性）
- 风险（锁定 / 误配 / 不可预测负载）

## 13. 优化建议矩阵（含节省估算）
```
建议 | 分类 | 影响(高/中/低) | 复杂度 | 预估节省(相对: 高/中/低) | 依赖 | 备注
```

## 14. 行动优先级与路线图
按阶段：立即 (0-2周) / 近端 (1-2月) / 中期 (季度) / 长期

## 15. 风险、假设与待验证项
- 风险登记：风险 | 影响 | 概率 | 缓解
- 假设清单：`假设: ...`
- 数据缺口：需进一步采集的项

## 16. 追加自动化 / 监控建议
