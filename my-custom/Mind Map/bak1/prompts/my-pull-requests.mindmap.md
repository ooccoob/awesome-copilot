# My Pull Requests Prompt - 思维导图

## 📋 文档摘要

**功能定位**：自动列出当前仓库中分配给我的 Pull Request，并分析状态与建议。

**适用场景**：需要快速了解自己负责的 PR 进展、评审和 CI 状态时。

**核心价值**：提升协作效率，及时发现待处理和有问题的 PR。

**使用指南**：自动检索、分析、汇总 PR 状态，突出待评审和失败项。

## 🎯 实际使用说明示例问题

- 如何快速列出分配给我的 PR？
- 如何分析 PR 的评审和 CI 状态？
- 如何请求 Copilot 评审？
- 如何定位有问题的 PR 并给出修复建议？

- PR Listing (PR 列表)
- Review Status (评审状态)
- CI Check Analysis (CI 检查分析)
